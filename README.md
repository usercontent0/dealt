# dealt (WIP)
A ROBLOX framework for making casino-style table games. dealt provides core game logic, state management, and event handling for games like Blackjack, Texas Hold'em, and Spades. Originally created to be used for the [DIMENSIONS](https://discord.gg/mTAREEz4yJ) project.
